# 🏀 NBA Match Outcome Predictor
A machine learning project designed to predict NBA game winners by analyzing historical player stats, team fatigue, and betting odds.

🚀 Key Features
- Automated Data Pipeline Custom Python script to scrape daily match data and player performance.
- Feature Engineering Integrated fatigue factors (back-to-back games) and home/away court advantages.
- Live Dashboard Built with **Streamlit** to visualize predictions and historical accuracy.

### 📊 Model Performance
| Model         | Accuracy | Precision 
|               |          | 
| Random Forest | 72%      | 68% 
| XGBoost       | 74.5%    | 71% 

🛠 Tech Stack
- Python (Pandas, Scikit-learn, BeautifulSoup)
- Streamlit (UI/Dashboard)
